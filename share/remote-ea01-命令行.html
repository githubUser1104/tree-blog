# linux目录结构

从根目录/开始

  • bin 存放二进制可执行文件(ls,cat,mkdir等)
  • boot 存放用于系统引导时使用的文件
  • dev 存放设备文件
  • etc 存放系统配置文件
  • home 存放用户文件，通常以用户名作为目录名再区分，例如/home/zhangsan
  • lib 存放跟文件系统中程序运行所需要的共享库及内核模块
  • mnt 系统管理员安装临时文件系统的安装点，例如可以将U盘挂载到该目录
  • opt 额外安装的可选应用程序包所放置的位置
  • proc 虚拟文件系统，存放当前内存的映射
  • root 超级用户目录
  • sbin 存放二进制可执行文件，只有root才能访问
  • tmp 存放各种临时文件
  • usr 存放系统应用程序，其中/usr/local为本地管理员软件安装目录
  • var 存放运行时需要改变数据的文件

用户目录~，默认是root目录

# 命令基础

cmd [options] [arguments]

  • 命令区分大小写
  • tab提示补全
  • 上下箭头查看历史命令
  • 别名alias（例如rm默认使用-rf参数） alias rm='rm -rf'
  • 空格间隔（例如同时删除多个指定的目录和文件） rm -rf test index.bak index.html
  • 支持正则

# 文件与目录管理

  • 查看当前目录 pwd
  • 切换目录 cd /usr/local
    • 切换到上一级目录 cd ..
    • 切换到当前目录下的local目录 cd ./local 或 cd loacl
    • 切换到cd跳转目录前所在的目录cd -
  • 显示当前目录下文件信息
    • 只显示文件名 ls
    • 详细信息 ls -l
    • 列出指定目录下的所有以w开头的文件或目录 ls /usr/bin/w*
  • 修改文件所属和权限(读写执行)
    • 利用数字 chmod 777 ./test.sh
    • 利用字母 chmod u=rwx,g=rx,o=r ./test.sh
  • 查看文件详细信息 stat test.md
  • 新建
    • 新建文件夹 mkdir -p /tmp/a/b/c
    • 新建文件 touch test.html
  • 删除指定文件或文件夹 rm -rf xxx
    • 删除该目录下所有文件 rm -rf *
  • 剪切文件
    • 把该目录下所有文件放到上一级目录 mv * ../
    • 改名文件 mv index.html index.bak
  • 复制文件 cp /root/index.html /usr/test/index.bak
    • 通常加上-a来支持复制文件夹和权限 cp -a /root/public/* .
    • 强制复制不询问是否覆盖(cp前面加上了\是因为在某些服务器 cp 被 alias命令 设置成 cp -i) \cp -rf * /etc/www/
  • 在当前目录下查找文件 find -name xxx
    • 在指定目录下查找文件 find /tmp/test -name xxx
  • 查看文件 cat index.html
    • 查看文件头10行 head -10 test.html
    • 查看文件尾10行 tail -10 test.html

# 磁盘管理

  • 查看所有磁盘使用情况 df -h
    • 查看当前目录所在磁盘的使用情况 df -h ./
  • 查看目录或文件的占用空间大小
    • 当前目录的大小 du -sh
    • 当前目录以及子目录的大小 du -lh | sort -n
    • 指定文件的大小 du -sh xxx.html

# 未分类

  • 重定向
    • 把左边的输出放到右边的文件（有则覆盖，无则创建）里去 echo "my content" > test.txt
    • 把左边的输出追加到右边的文件（有则追加，无则创建）里去 echo "my content" >> test.txt
  • 管道
    • 将前面命令的结果给后面的命令（将ll的结果加入wc统计字数） ll | wc
    • 通过管道和grep命令搜索输出的结果中的关键字，例如搜索网络信息中的ip ifconfig | grep inet
  • linux自带解压（将nginx源码包解压到当前目录，v代表输出过程） tar -xvf nginx-1.20.1.tar.gz
  • linux自带压缩（将当前目录的文件压缩成public.tar.gz） tar -cvf public.tar.gz *
  • centos自带包管理 yum -y install xxx
  • 检测网络连通性 ping ip或www.baidu.com
  • 防火墙 (opens new window)
    • 防火墙状态 systemctl status firewalld.service
    • 关闭防火墙 systemctl stop firewalld.service
    • 禁用防火墙 systemctl disable firewalld.service
  • 进程（例如nginx）
    • 查看nginx进程号 ps -ef | grep nginx
    • 查看xxxx端口的进程 lsof -i :xxxx
    • 结束进程 kill -9 进程号pid
  • 查看端口占用：netstat -ntulp | grep 3306
  • 查看网络信息：ip ifconfig
  • shell环境变量：略

  • 安装 yum install zip unzip

  • 解压 unzip test.zip

    • 将dist.zip解压到指定路径，编码支持中文 unzip -O cp936 dist.zip -d .

# 语法

  • 特性
    • 文件测试运算符
  • 示例
# 定义变量，注意等号左右不能有空格
name="zhangsan"
# 使用变量
echo "I am ${name}"
# 字符串，单引号原样输出，双引号可以使用变量和转义
str1='this is a string'
str2="Hello, I know you are \"${name}\"! \n"
echo ${str2}
# echo 如果是反引号则显示命令执行结果
echo 'date'
echo `date`
# 算数运算
a=10
b=101
val=`expr 2 + 2`
echo "2+2为 : $val"
result=$[a+b]
echo "a+b 为： $result"
# 条件语句 if then (else) fi
if [[ $a -lt 100 && $b -gt 100 ]]
then
   echo "a小于100且b大于100"
else
   echo "返回 false"
fi
# 选择语句 右括号和双分号是特性
site="runoob"
case "$site" in
   "runoob") echo "菜鸟教程";;
   "google") echo "Google 搜索";;
   "taobao") echo "淘宝网";;
esac
# 循环语句 for in do done
for loop in 1 2 3 4 5
do
    echo "The value is: $loop"
done
# 函数
funWithReturn(){
	# 如果需要获得第10个及以后的参数，要用花括号 ${10}
	a=$1
	b=$2
  return $[a+b]
}
echo "-----函数开始执行-----"
funWithReturn 3 5
# 最近的返回值通过 $? 获取
rs=$?
echo "输入的两个数字之和为 $rs"
echo "-----函数执行完毕-----"
